Spring-boot-x开发日志

Sep 18, 2020 16:35   |   spring   |   #spring   |   One minute read

起源

​ 一月前，做了这样的一个东西，这个东西目标是打破SpringBoot的Application启动时，在不指定扫描包路径时，扫描启动类所在的包及其子包。实现这个，就可以完成简单的引入一个外部的jar，这样就可以在这个外部的jar中来扩展各种功能了。在这样的情况下，spring-boot-x就一点点的诞生了。

​ 说到一点点的诞生是因为在开发过程中，对它的源码和设计经历了从复杂到简化，从代码多到代码少一点点的不断的改进。

支持的功能

  1. pretty请求日志
  2. 自定义Error的异常处理，Error View
  3. Redis Command风格的RedisTemplate
  4. Redis Keynaming
  5. feign header 包装
  6. TODO
  • Kong系列之入门(上)
  • 《一往无前》读后感
